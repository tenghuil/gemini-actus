# Gemini Actus Companion

The Gemini Actus Companion extension pairs with
[Gemini Actus](https://github.com/google-gemini/gemini-cli). This extension is
compatible with both VS Code and VS Code forks.

# Features

- Open Editor File Context: Gemini Actus gains awareness of the files you have
  open in your editor, providing it with a richer understanding of your
  project's structure and content.

- Selection Context: Gemini Actus can easily access your cursor's position and
  selected text within the editor, giving it valuable context directly from your
  current work.

- Native Diffing: Seamlessly view, modify, and accept code changes suggested by
  Gemini Actus directly within the editor.

- Launch Gemini Actus: Quickly start a new Gemini Actus session from the Command
  Palette (Cmd+Shift+P or Ctrl+Shift+P) by running the "Gemini Actus: Run"
  command.

# Requirements

To use this extension, you'll need:

- VS Code version 1.99.0 or newer
- Gemini Actus (installed separately) running within the integrated terminal
